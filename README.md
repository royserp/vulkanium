# Vulkanium

> ⚡ A high-performance rendering and optimization mod for Minecraft
> Built for speed. Designed for stability.

**Vulkanium** is a next-generation rendering optimization mod for the Minecraft client, focused on delivering higher frame rates, reduced micro-stutter, and improved graphical consistency across a wide range of hardware.

> 🔧 Vulkanium is a **fork of Sodium**, extended and maintained by **theroyalserpant**.

---

## 🚀 Features

- ⚡ Significant FPS improvements
- 🧠 Reduced stuttering and smoother frame pacing
- 🛠️ Engine-level rendering optimizations
- 🎮 Better performance on low-end and mid-range systems
- 🔬 Experimental rendering enhancements (ongoing)
- 🔥 Vulkan-based rendering *(launched)*

---

## 📊 Why Vulkanium?

Minecraft's default renderer is not optimized for modern hardware. Vulkanium improves performance by:

- Reducing CPU overhead
- Optimizing chunk rendering pipelines
- Improving memory usage patterns
- Minimizing unnecessary rendering calls

Result: **Smoother gameplay, higher FPS, better experience.**
